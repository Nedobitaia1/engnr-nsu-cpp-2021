# Сборка программы:

1) `cmake -G "MinGW Makefiles" ..`
2) `mingw32-make`

# Запуск программы:
`2 <filepath>` или `2`

# Существующие команды в калькуляторе:
1) `# - комментарий`
2) `POP`
3) `PUSH <number>`
4) `PUSH <varname>` или `PEEK <varname>`
5) `ABS`
6) `PLUS`
7) `MINUS`
8) `MUL`
9) `DIV`
10) `PRINT`
11) `READ`
